# 🎓 Student Performance Analysis Using Python & Power BI

## 📌 Project Overview

This project analyzes student academic performance using Python, Pandas, Matplotlib, Seaborn. The analysis is based on a dataset containing **1,000 student records** and explores the impact of gender, parental education, lunch type, and test preparation courses on student performance in Mathematics, Reading, and Writing.

The goal of this project is to identify key factors influencing academic achievement through Exploratory Data Analysis (EDA), statistical visualizations, feature engineering, and interactive dashboards.

---

## 📊 Dataset Information

- **Total Records:** 1,000
- **Total Features:** 8
- **Dataset Type:** Educational / Student Performance Data

### Features

- Test Preparation Course
- Math Score
- Reading Score
- Writing Score

## ⚙️ Feature Engineering

Created a new feature:

```python
Total Score = Math Score + Reading Score + Writing Score
```

This feature was used to identify top-performing students and compare overall academic performance.

## 💡 Key Insights

- Female students performed better in Reading and Writing.
- Students who completed the test preparation course scored significantly higher.
- Students with standard lunch generally achieved better academic results.
- Reading and Writing scores showed a strong positive correlation.
- Higher parental education levels were associated with improved student performance.
